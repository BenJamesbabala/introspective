============
ML Insights
============

Package to understand ML Models

Installation:
-------------

.. code-block:: bash

    $ pip install ml_insights


Usage:
------

.. code-block:: python

    >>> import ml_insights as mli
    >>> xray = mli.ModelXRay(model, data)
